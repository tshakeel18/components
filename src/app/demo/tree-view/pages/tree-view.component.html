Simple Usage

Select items by clicking on them. cmd/ctrl + click to multiselect. Multiselect is disabled by default.

Add Child Nodes

Click the icon to expand and collapse the node or just double click the name.

Change the default icons of the folders using the iconFolderOpened and iconFolderClosed options.

Single Expanded Node

Only let one node to be expanded at a time by using the input singleExpanded.

Disabled Nodes

Cannot interact with them.

Set Your Favorite Icons

This is for the nodes without children.

Listen For Selection Events

selected and deselected events are emitted when an item gets toggled.

Access Nodes From Outside

Add a #reference to the element and then access the nodes in your code at reference.nodes.

Custom Folder icons

Click on the folder with custom icons.

Flat Folder listing

Click on the folder to view flattened hiearchy.

The folders are flattened by setting collapseFolders to true. The separator can be customized by setting collapseFolderSeparator as ' > '.

Custom Templates

You can define your own template to display custom content inside any particular node.

Nodes with handlers

Enabling this feature will display handlers alongside the folder icons. This is useful in scenarios where the folder icons don't change based on its expanded/collapsed state.

Lazy loading nodes

Children nodes can be lazy-loaded by implementing the DfTreeViewLazyNode interface.
